# 第二十三章 先行经过的手

结构体内侧激活的引导反应很弱，而最近的脱离痕迹却很明显。

如果只有其中之一，判断会很简单。如果只剩引导反应，就会继续往更深处确认；如果只有最近的痕迹，就会直接追踪。问题是现在两者同时存在。剩余的路径还没有死，而先行取走更大碎片的某人也并非很久之前消失的。

希昂最先开口。

"做决定吧。先看更深处，还是现在咬住尾巴。"

希昂没有立刻回答。目光仍在结构体内侧的黑暗与槽位外侧的最近痕迹之间来回游移。取走更大碎片的人留下的重心转移痕迹、被细微刮擦过的金属面，以及极其匆忙转身离开时扭过一次身体的痕迹。那既是逃跑者的痕迹，同时也是能读懂这个结构的人的痕迹。

瑟恩低声说道。

"如果要追踪，现在是距离最近的时刻。再晚的话，尘层和振动会抹杀痕迹。"

阿特尔几乎同时指出了另一面。

"但如果更深处残留的反应断掉，重新打开这整个结构体的代价可能会变大。"

韩智宇从船体那边低声说道。

"两句都没说错。所以更讨厌。"

希昂看着内侧的黑暗，非常缓慢地说。

"两个都拿不了。"

"当然拿不了。"
希昂低声说。
"所以才说现在做决定啊。"

瑟恩再次查看了最近脱离痕迹附近。尘层断过一次，细微的金属粉末向外侧散落。如果是匆忙取走东西的人，重心应该会有更大的摇晃，但这痕迹出乎意料地整齐。虽然赶过，但并未失控的动作。也就是说，是熟悉现场的人。

他非常低声地说。

"对方至少不是业余的。是急但不乱的人的痕迹。"

阿特尔再次看向最近的刮擦痕迹。取出更大碎片的位置、碎屑卡住后残留的角度、槽位边缘的损伤程度。没错。这不是不了解结构的人匆忙拆下的痕迹。这至少是理解过一次这种逻辑之后才动的手。

他安静地说。

"那么先行经过的人是能读懂这个结构的一方。"

希昂没有立刻对这句话做出反应。不是单纯的偷窃者，而是越过了判别、精准取走更大碎片的人。那么这就不是偷东西的问题，而是谁先读懂了这条路的问题。

近距频道那边，瑟琳的声音短促地传了进来。

"那就更应该现在追。"

阿特尔抬起了目光。

"依据是。"

"里面的东西还会在，先行经过的手不会在。"
瑟琳干脆利落地说。
"结构可以重新读。人再错过就没了。"

短暂的静默。

希昂听完那句话，嘴角极轻地上扬了一点。瑟琳说得对。内侧的反应虽然不稳定，但还活着。可人一旦再远一步，在这片外围星团这边就会直接融进宇宙里消失。

他终于抬起了头。

"追。"

阿特尔没有再问。那个决定看上去像是感性的，实际上却是最现实的选择。而自己也找不到反对这个选择的理由——这个事实既令人不快，又格外清晰。

瑟恩立刻问道。

"行进顺序。"

希昂用手指指向最近痕迹延伸的方向。

"脱离的那一方没有直接往外撤。先往内侧结构体的阴影方向藏了一次身，再从那里转向外部接驳带方向。"

韩智宇立刻接上。

"那我把船体再往外拉一点，做出回收船的角度。"

"不。"
希昂摇了摇头。
"智宇，你保持船体。万一谁再靠上来，我们需要回去的船。"

韩智宇没有反驳。只是近距频道那边传来了一声极短的呼吸。

近距频道那边传来瑟琳的声音。

"谁打头。"

"我。"
希昂说。
"瑟恩居中。阿特尔在后面读结构。"

稍后，瑟琳短促地说。

"要是硬撑，我先断。"

希昂目视前方，回答道。

"知道。"

短暂的对话结束后，三人极其迅速地行动起来。希昂率先沿着最近的重心转移痕迹转向了结构体外侧。路并非路的形状。被切断的环扣与断裂的踏板、外部校准部件的残骸、船坞维护用骨架半露出来的狭窄缝隙。这不是通道，而是只有了解结构的人才能勉强通过的连接部。也就是说，先行经过的人也读得懂这条路。

瑟恩在后面立刻说道。

"左侧踏板，承重已死。"

希昂听到话的瞬间就调转了脚步。紧接着下一刻，刚才正要踩上去的金属板下方无声地塌陷了。与其说是坠落，更像是一直在硬撑的东西自行崩溃时的那种沉默。

近距频道那边，瑟琳压低了呼吸。

"一个人来的话早就掉下去了。"

"一个人的话根本不会来。"

"那是假话。"

希昂没有接话，直接看向前方。

结构体阴影内侧的壁面上，这次留着一道清晰得多的痕迹。金属表面上像是手背蹭过的灼痕，旁边还有一截极短的断裂纤维碎片。看起来像是外套下摆或手套内衬被刮擦后留下的痕迹。

瑟恩伸手极其小心地捡起了那片纤维。

"是新的。"

阿特尔一看到就目光沉了下来。

"不是帝国制式的。"

希昂立刻问。

"同盟这边的？"

阿特尔无法确定。

"至少不是承认院规格的。也不是港口工人用的。"

这是最让人不舒服的那种。不是帝国，不是同盟，也不是底层的杂牌商贩的手。是从权力结构之外读懂结构、先行进入的灰色之手。

希昂看纤维碎片的时间不如看它旁边金属面上残留的手背痕迹长。

"那不是跑的时候蹭到的。是停下来撑住的痕迹。"

阿特尔安静地接过话。

"如果是那样的话，说明对方也在这里读过一次方向后才重新行动的。"

希昂非常缓慢地指向前方。

"那边。"

结构体外侧阴影终结的位置，外部接驳带残骸与老旧的校准板交叠的部分。是某人藏身之后再次出去的最合适的位置。

现在独立的目标已经明确了。继续深入读取内侧反应也很重要，但眼下更紧迫的是不要放跑带着更大碎片先行经过的那只手。残留的反应很弱，痕迹也在迅速消亡。而能在此刻、在这个位置直接读取现场并接续下去的，终究只有他们。

四人同时压低了呼吸。

就在那时，前方的黑暗中传来金属极浅地摩擦的声音。

仿佛在说，有人还没有走远。
