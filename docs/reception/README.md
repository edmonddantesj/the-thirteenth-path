# Public Reception Log

This folder collects selected public reader responses to **The Thirteenth Path**.

It is not a raw comment archive. It is a curated reception log: short excerpts, summaries, and links to public source comments that help show how readers are actually receiving the story.

## Purpose

- Preserve strong reader responses without exposing private chat context.
- Track how each episode, arc, character, and language edition is being read.
- Build qualitative proof for future partner, publisher, press, and product packets.
- Keep a rights-safe record that can be reused without copying full third-party comments into the repository.

## Publication Rules

Only add a reader response here when all of the following are true:

- The source is publicly accessible, or the reader has explicitly allowed public use.
- The entry uses a short excerpt or an author-written summary, not a full private conversation.
- Personal details, private handles, message IDs, and non-public context are removed unless explicit permission exists.
- The entry links back to the original public source when a stable public URL exists.
- The response is presented as reception evidence, not as an endorsement fabricated by AOI.

Default policy: anonymize.

Use a public handle only when the comment was already made under that public handle and the source URL is included, or when the reader explicitly asks to be credited.

## What To Capture

Prioritize responses that reveal how the work is being read:

- literary response to names, records, routes, order, thresholds, and responsibility;
- episode-specific emotional or comprehension reactions;
- character attachment or tension;
- language/translation reception;
- music, Codex, or app-reading experience;
- AI-agent collaboration understood as literary context, not only novelty;
- confusion or friction that teaches AOI how to improve the surface.

Do not prioritize comments merely because they say the work is "AI" or "interesting." The important question is what the story did to the reader.
