# 第七十話　まだもっと生かさなければならない体

補助結合部に初めて火がついた後も、
内側の作業はそこでは終わらなかった。

終われるはずがなかった。

外でハルンの精算が済み、
スクラップ市場の空気が一度沈む間にも、
内側ではほぼ一晩まるごと使い切っていた。

シオンは手の中の火種を少し引いた状態でも、
たった今船体の内側で短くつながったあの微かな震えをずっと思い返していた。

本当に短かった。
本当に小さかった。

だからこそもっとはっきりしていた。

偽物ならもっと華やかだったはずだ。
音ももっと大きく、光ももっと目立ったはずだ。
だがたった今起きたのはそういう類ではなかった。

あまりに小さいからこそ、
本当にこの船体のどこかがまだ通っていた方式を覚えている側に近かった。

ハン・ジウはもう次を見ていた。

補助結合部の内側と、
まだ触れていない中心側の構造、
そして受ける枠に残った微かな熱の跡を交互に見た。

「ここからすぐ先に行っちゃだめだ」
低く言った。

シオンが訊いた。

「なぜ」

「ついたのと堪えるのは違う」
ハン・ジウが答えた。
「さっきは押し返さなかったってだけで、ずっと耐えるって意味じゃない」

アカがその言葉の上に静かに付け加えた。

「今もっと食わせたらまた死ぬ」

短い文。

だがシオンにはその言葉がほとんど体で理解できた。
さっき手をほんの少しだけ下ろしたときも、
受ける枠と補助結合部の間の緊張がぱっと変わった。ごく小さな差だったのに、これ以上はだめだという感覚は確かにあった。

なら今必要なのは成功の場面をもっと大きくすることではなく、
たった今蘇ったあの小さなつながりが冷めないよう堪える時間を与えることだ。

ハン・ジウは船体の下にもっと体を屈めた。

「補助結合部は生かせる」
ハン・ジウが言った。
「でも中心まで引っ張るにはここだけじゃ足りない。重みを受ける別の軸も作らないと」

アカはうなずいた。

「それと同じ結がもっと要る」
彼女が言った。

シオンにはすぐ分かった。

濁って不細工で長く堪えたあのエテライトの欠片のようなもの。
今のように小さな火をまず受けてくれる材料。
何でも代わりに突っ込めるものではないもの。

「たくさんないのか」
シオンが訊いた。

アカは短く答えた。

「ここの中には少ない」

それは事実上、外でもっと手に入れなければならないという言葉だった。

ハン・ジウも同じ結論にいた。

「受ける枠をもう一つ作らなきゃいけない」
ハン・ジウが言った。
「それに外皮の金属も替えるべきところが見える。今は内側が堪えるとしても、船体を動かすレベルになったら外が先に裂ける」

シオンは船体をもう一度見た。

昨日まではただ半ば死んだ体のように見えたものが、
今はどこが先に堪えてどこがまだ堪えられないかが少しずつ見えていた。

だからこそもっと分かった。

これは一日二日で終わる作業ではない。
たった今やったことは始まりであって、
完成ではない。

アカは受ける枠を慎重に外さず、
その場所に仮固定する方を選んだ。ごく弱い結合糸と薄い金属片を添えて、たった今蘇った最初の反応がすぐ散らないように支える形だった。

「これは残す」
彼女が言った。
「冷めるか見る」

ハン・ジウはその言葉にすぐうなずいた。

「そうだ」
ハン・ジウが言った。
「今大事なのはもっと押し込むことじゃなくて、これが本当に堪えるか見ること」

シオンは手の中の火種をもう一度包んだ。

この火も休ませなければという考えが初めて浮かんだ。
ずっと握ってはいたが、
ずっと何かに当ててみようとする気持ちそのものが
むしろもっと危ないかもしれなかった。

アカがシオンの方を見た。

「あんたも」
彼女が言った。

シオンが問い返した。

「何を」

「休んで」

シオンは一瞬言葉を失った。

その言葉が不思議だった。
ハズランに入ってから今まで、
誰かに自分がそう言われたことがほとんどなかったからだ。大抵はもっと走らなければならず、もっと堪えなければならず、もっと速く判断しなければならなかった。

だがアカはまるでそれも作業の一部であるかのように、
休むことをあまりに当然のように言った。

ハン・ジウがその横でごく低く笑った。

「正しいな」
ハン・ジウが言った。
「今あんたの手の感覚が死んだら、さっきついたのも次はつけられなくなる」

シオンはその言葉に結局短く息を吐いた。

そうだ。

今必要なのは勇敢にもっと押すことではなく、
次にまた正確につけられる状態を残すこと。

アカは船体から一歩引いた。

そして初めて、
今日の中ではもうこれ以上は行かないという人間のように言った。

「ここまで」

短いが確かな宣言だった。

シオンはその言葉を聞いた瞬間、
内側の作業が止まったことより
もう外が必要になったことの方を先に感じた。

もっと多くの材料。
もっと合う金属。
時間。
そして外で生きている人々の手。

ハン・ジウももう同じ考えだった。

「部品をもっと見ないと」
ハン・ジウが言った。
「エテライトももっと要るし、船体の外皮の代わりになる金属も選ばなきゃ」

アカは短く言った。

「スクラップ市場にある」

シオンはその言葉を聞いてごく微かに笑いそうになった。

結局また外だ。
だが今度は追い出されるのではなく、
生かすために出る。

その違いは大きかった。

内側の敷居を出る直前、
シオンは一度振り返った。

補助結合部に仮固定された最初の受ける枠。
その内側にごくかすかに残っている微かな熱。
そして完全に死んだ体ではないという言葉をたった今聞いた船体。

まだ遠い。
だが完全に空想の話でもない。

その感覚一つを持って、
三人は再び外へ出た。

外の空気は内側より荒かったが、
もうその荒さも前のようには途方もなくなかった。

やるべきことがはっきりしたからだ。

ハズランはまだ発つ場所ではなく、
もう少し留まってこの船を生かすものを集める場所になった。
