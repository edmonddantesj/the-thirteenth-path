# Version Registry

This registry tracks the public GitHub archive status for **The Thirteenth Path / 열세 번째 길**.

It is intentionally lightweight. The private setting bible and production archive may contain more detailed revision history.

## Version Key

- **KO canonical** — Korean public/canonical file version in this repository.
- **EN public** — English public file version in this repository and the Korean version it is based on.
- **JP public** — Japanese public file version, if any.
- **Continuity impact** — how strongly later canon depends on this episode.
- **Retcon risk** — estimated risk if the episode is revised later.

## Update Rule

When a public episode is revised, update the relevant row:

```text
Episode: EP035
KO canonical: v1.1
EN public: v1.0 based on KO v1.0 / needs sync
JP public: pending
Continuity impact: High
Retcon risk: Low
Notes: short reason for revision
```

If a translation is updated from a newer Korean source, record the source version clearly.
